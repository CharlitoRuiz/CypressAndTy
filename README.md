# CypressAndTy
Test hechos con Cypress y Typescript

Comandos para crear el proyecto:

npm init

instalar cy:
npm install cypress --save-dev

npx cypress open

instalar typescript
npm i typescript

npx tsc --init

abrir cy
npx cypress open

cli mode
npx cypress run --spec="./cypress/integration/specs/google-search.spec.ts" 

xpath por si se necesita
npm i cypress-xpath
